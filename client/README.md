# POS (Next.js)

This project was converted from a Vite + React (react-router-dom) app to **Next.js App Router**.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

> Note: Your original **React Router** routes are kept as-is (SPA mode).  
> Next.js serves `/[[...slug]]` so refreshing deep links (e.g. `/dashboard`) works.
